# Firefox DevTools Theme (VS Code)

A Visual Studio Code theme inspired by Firefox DevTools.

## Themes

- Firefox DevTools Dark
- Firefox DevTools Light

## Install

### From Marketplace

Search for `Firefox DevTools Theme` in the Extensions view.

### From VSIX

1. Download/build the `.vsix`
2. In VS Code: Extensions -> `...` -> `Install from VSIX...`

## Development

- Build VSIX: `npm run package`
